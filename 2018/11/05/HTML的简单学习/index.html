HTML的简单学习

最近学习了HTML 今天抽空一下整理 ， 方便日后查看。

HTML介绍

HTML（hyper text markup language）：超文本标记语言。超文本，不止文本，还有图片、视频、下载内容、用户交互等。

标记语言：编程语言有逻辑控制if else、变量、方法、类。标记语言相比较为简单，由信息和结构语法构成。

历史：html4 将要学习的内容。在html4的基础上，添加新功能和语法，就是html5，h5，但是95%都是一样的。好像py2升级到py3一样。xhtml是一种语法严格版的html，但是用的不多。

###＃　(了解)环境准备

  • 推荐谷歌浏览器。其它360、遨游、微软edge、微软ie、火狐、opera。
  • 编辑器：pycharm pro或sublime

HTML 基础介绍

标签：<> 括住的叫标签，不同功能和结构。一般成对儿出现 开始<xxx> 结束关闭</xxx>，标签中可以填写内容。标签可以没有闭标签。
属性：标签内键=值的形式，键叫做属性，属性可以设置标签的行为。
元素：标签加标签内的内容。

标签可以嵌套，代码风格 每一级标签缩进2个空格或4个空格。

<html>: 网页的根节点

几个关于文字效果的标签：

b标签 ：bold加粗
i标签： italic 斜体
em: emphasize text 强调文本
strong： 强调文本
small： 弱化文本 。字小颜色浅字体细。

概念：

标签：形如<xxx> </xxx>,<xxx>是标签开始 </xxx>标签结束。标签中填写内容。一些不要内容的标签，例如<hr> <br>,省略闭标签。标签有自己的特定功能，定义html的具体结构。
属性：写在标签中，定义标签的具体行为或额外的设置。一个标签通常有多种属性，例如id class background-color align等。属性键=”值”，跟python里的dict类似。

常用属性：

style: 样式。style="background-color: yellow"
align： 字体对齐方式。 align="center"

HTML 表格

  • table标签: 表格。
  • tr标签： table row 表格中的一行。
  • th标签： table head 表头
  • td标签： table data 表格数据一格

属性：

  • border： 边界
  • align： 表格对齐方式
  • bgcolor： 表格背景颜色
  • background： 背景图片
  • cellpadding: 一格内文字距离边界的距离
  • cellspacing: 格与格之间的距离。
  • colspan : 列合并
  • rowspan ： 行合并

HTML 图片

  • img标签：image 图片。
    属性:
  • src：source 资源路径，必写项。可以是本地项目路径，也可以是互联网上的路径。
  • width： 宽 。默认单位是px像素。长度单位还可以写成百分比形式。
  • height： 高。默认单位是px像素。长度单位还可以写成百分比形式。

HTML 列表

ol标签： order list 有序列表
li标签： list item 列表中的每一项
ul标签： unorder list 无序列表
type属性（了解）：列表前可以是数字或字母或方形点。

###　ＨTML 表单

form标签：表单。

input标签：用户输入相关。
type属性：

  • text 文本框
  • password 密码框 ，自动隐藏用户输入内容
  • radio 单选。注意那一组input选项起相同的name属性，写好value属性。
  • checkbox 多选。
  • file 上传文件 。 注意form头部声明enctype属性
  • reset 重置
  • submit 提交
  • action属性： 定义表单请求的后台地址。
  • required属性： 必填项 true
  • readonly属性: 只读 readonly

select标签：下拉列表。
option标签：下拉列表中的一项。

textarea标签： 文本域，比较长、大块文本。
